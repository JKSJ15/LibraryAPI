# Books API - Spring Boot

REST API developed with **Spring Boot** for managing books, providing full CRUD operations, pagination, and dynamic filtering.

---

## Technologies

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* Maven
* Relational Database (MySQL or PostgreSQL)

---

## Features

* ✅ Full CRUD operations
* ✅ Pagination support
* ✅ Dynamic filters (title, author, genre, publication date)
* ✅ Data validation with Bean Validation
* ✅ Global exception handling
* ✅ Use of DTOs for request/response

---

## Endpoints

### Get all books (with pagination and filters)

```
GET /books
```

### Optional query params:

* `title` → filter by title (partial match, case insensitive)
* `author` → filter by author
* `genre` → filter by genre
* `date` → filter by publication date
* `page`, `size`, `sort` → pagination and sorting

---

## Validations

* Required fields (e.g., title and author)
* Publication date cannot be in the future
* Rules defined using Bean Validation (`@NotNull`, etc.)

---

## Business Rules

* ID is automatically generated by the database
* ID is not required when creating a book
* Text search is partial and case insensitive

---

## How to Run

1. Clone the repository:
```
git clone https://github.com/your-username/books-api-spring-boot.git
```
2. Navigate to the project folder:
```
cd books-api-spring-boot
```
3. Configure the database in `application.properties`
4. Run the application:
```
./mvnw spring-boot:run
```
